4letterwords.

 Aug 20, 2015

Summer 2015, I started getting passionate about how I could use advertising to change the world. After trying to build an alarm clock that woke people up with video ad’s, I decided to make a mobile trivia game. I took an archive of Jeopardy Questions and filtered out all answers that were not four letters long. Ads only appear if you run out of points. Points are gained by typing in the answers correctly. Points are used to buy hints, which give you one of the letters needed for the correct answer.

Built with:
cordova
phonegap
materialize
Java
JavaScript
Android
iOS
