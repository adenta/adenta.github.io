Setting up a Remote Repository

This section of the tutorial assumes your git repo looks like this.

Creating the repo

Now that we have made a commit to our repository in a new branch, let’s find a place to put it. Log on to whatever service you use to manage git repos (Github, Gitlab, Bit Bucket, etc.). If you are an IU student, you will probably be using IU Github.

For this tutorial, we will assume you are using github.

When you arrive at the page, hit the ‘create new’ button in the upper right corner. From here, hit ‘New repository’.

At this screen, give your repository a name, and hit ‘Create Repository’. Now you will be on a page that looks like this.

Security Config

Leave this page open, and head back into your terminal. Use these documents to setup SSH keys. We will be adding more documentation on this step soon, please reach out if you have any questions.

Once the keys are added, you should be ready to push your repo to Github!
